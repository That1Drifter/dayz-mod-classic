Install
troubleshooting

Common problems hitting first-time players, and how to fix each one.

Installer says Arma 2 OA not detected

The installer looks for Arma 2: Operation Arrowhead in your Steam library. If it can't find it:

  1. Install Steam if you haven't already.
  2. Buy and install both Arma 2 and Arma 2: Operation Arrowhead on Steam.
  3. Launch Arma 2: Operation Arrowhead once from Steam to confirm it works.
  4. Re-run DayZModClassic-Client-1.0.0-Setup-v3.exe.

If you believe A2OA is installed and the installer still fails, share the diagnostic file it writes to %TEMP%\DayZModClassic-install-diag.txt. Drop it in Discord and we will help.

Launcher says "Missing addons"

Arma 2 is not installed or Steam returned a bad path. Make sure both Arma 2 and A2OA are fully installed via Steam, then re-run the mod installer. The installer can safely run on top of an existing install; it repairs missing files.

Launcher says BattlEye fix not present

The installer drops the Win11 24H2 BattlEye compatibility files. If A2OA was missing at install time, this step is skipped.

Fix: re-run DayZModClassic-Client-1.0.0-Setup-v3.exe after A2OA is fully installed.

Connect fails: Wait for host

The server is empty and booting on first connect (the official server restarts lazily to save resources).

  1. Wait about 30 seconds.
  2. Click PLAY again from the launcher.
  3. If it keeps failing, check status.json to confirm the server is online.

Connect fails: Player without identity

Steam isn't running, so Arma can't load your Steam identity.

Fix: start Steam and sign in, then click PLAY again from the launcher.

BattlEye kicks on connect

The installer bundles the Dwarden Win11 24H2 BE fix. If kicks persist:

  1. Verify the installer completed without errors.
  2. Make sure Steam is running when you connect.
  3. Try re-running the mod installer to repair the BE files.

Windows says "Windows protected your PC"

The installer is unsigned (code-signing certs cost ~$200/yr for an indie project). To proceed:

  1. Click More info on the warning dialog.
  2. Click Run anyway.

You can verify the file is untampered by checking the SHA-256 hash on the downloads page.

Game crashes on startup

Open the Arma report log:

%LOCALAPPDATA%\ArmA 2 OA\arma2oa.RPT

Paste the last 50 lines of that file into the #help channel on Discord. Someone will help diagnose it.

Still stuck?

From the launcher, open Help ▾ → Save diagnostic report.... It writes a zip to your Desktop with launcher logs, the Arma RPT, your scrubbed config, and the installer diagnostic (if present). Drop the zip into our Discord and we will dig in.
