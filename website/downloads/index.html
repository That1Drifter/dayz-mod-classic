02

How to install

01

Buy both Arma titles on Steam

You need Arma 2 and Arma 2: Operation Arrowhead. Install both through Steam before running the mod installer.

02

Run the launcher, click INSTALL

Download DayZModClassic.exe (~69 MB, bundles its own runtime) and run it from anywhere. It detects your Steam library, then INSTALL downloads the mod files (~206 MB) and the BattlEye Win11 fix straight into your Arma 2 OA folder.

03

Connect

Select a server and click PLAY. Steam must be running. When a patch ships, the button reads UPDATE and only the changed files download.

// Windows SmartScreen warning

  2. Click Run anyway.

Troubleshooting

▸

Launcher won't run

If SmartScreen blocks it, click More info then Run anyway. If it opens and immediately closes, check for a second copy already running in the tray.

▸

Launcher says "Missing addons"

Arma 2 is not installed or Steam returned a bad path. Make sure both Arma 2 and A2OA are fully installed via Steam, then click INSTALL again in the launcher.

▸

BattlEye kicks on connect

The launcher installs the Dwarden Win11 24H2 BE fix with the mod. If kicks persist, check the launcher's STATUS panel shows all green and Steam is running when you connect.

▸

Still stuck?

See the full troubleshooting guide or ask in Discord.
