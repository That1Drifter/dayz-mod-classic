System requirements

  • Windows 10 / 11 (64-bit)
  • Steam account
  • Arma 2 + Arma 2: Operation Arrowhead, purchased on Steam (store page)
  • ~5 GB free disk space

How to install

  1. Buy and install Arma 2 and Arma 2: Operation Arrowhead via Steam.
  2. Download and run DayZModClassic-Client-1.0.0-Setup.exe.
  3. Click through the installer. It handles BattlEye fixes and mod files for you.
  4. Open "DayZ Mod Classic" from the Start Menu and click PLAY.
Windows says "Windows protected your PC", what do I do?

This is normal for indie software without an expensive code-signing certificate. To proceed:

  1. Click "More info" on the warning dialog.
  2. Click "Run anyway".

The installer is safe. You can verify the SHA-256 hash listed below matches the file you downloaded.
